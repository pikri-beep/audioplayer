# 🎵 Music player

Modern Desktop Music Player built with Electron.

## Features

- 🎵 Playlist
- ❤️ Favorite Songs
- 🖼 Custom Album Cover
- 🌈 Dynamic Ambient Color
- 🔀 Shuffle
- 🔁 Repeat
- 🎚 Volume Slider
- 📂 Import Local Music
- 🖥 System Tray
- 🎮 Windows Taskbar Controls
- 🌊 Crossfade Transition

## Installation

```bash
npm install
npm start
```

## Add Music

Copy your songs into

```
songs/
```

or click the **+** button inside the app.
